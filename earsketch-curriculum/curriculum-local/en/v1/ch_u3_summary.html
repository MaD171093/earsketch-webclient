Chapter 17:

  • Console input entrusts certain decisions about musical output to the user, making a script more interactive. 17.1

Chapter 18 and Chapter 19:

  • Data structures store a collection of values in a single entity. Indices are used to refer to individual elements of the structure. 18.1

  • A for-loop provides a simple way to iterate through an entire structure. 18.2

  • makeBeat() can be used with data structures to create rhythms out of multiple sound clips. 18.3

  • Data structures can be concatenated and sliced. 19.1

Chapter 20:

  • Randomness allows the computer to control some aspects of music making. The majority of random processes in EarSketch rely on the generation of random numbers, a number that is impossible to predict. 20.1

  • Introducing randomness into a program creates unique musical output and variations each time the code is run.
