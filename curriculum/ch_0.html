Welcome to EarSketch! In the following lessons, you will learn computer science and music technology side by side. You will use either Python or JavaScript, two of the most popular computer programming languages in the world, to create and remix music that is unique to you. Musicians and programmers write computer code to create new sounds, effects, and musical structures. Computer code enables you to design entirely new ways to create and perform music.

Learning to write computer code is a valuable skill that lends itself to a broad range of constantly growing, innovative careers. These opportunities extend into careers in music; EarSketch uses the same kind of digital audio workstation (DAW) software used throughout the music industry.
